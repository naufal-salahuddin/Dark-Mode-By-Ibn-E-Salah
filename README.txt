== Description ==

Dark Mode By Ibn_E_Salah. Use this shortcode [darkmode_switcher_by_Ibn_E_Salah] or show on front from setting section. 
It's a really simple plugin and doesn't add any CSS on its own. You can write your own custom css in
Customizer or from Plugin Settings.

== Installation ==

1. Upload `dark-mode-by-Ibn_E_Salah.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this plugin add predefined css =

No, This plugin doesn't add any css, you can add your custom css in theme customizer or in plugin settings.

= Does this plugin use Cookies? =

Yes this plugin use cookies to set dark mode.
